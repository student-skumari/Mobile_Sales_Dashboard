# Mobile_Sales_Dashboard
📱 Mobile Sales Dashboard -

Mobile Sales Dashboard | Power BI

Designed and developed an interactive Mobile Sales Dashboard in Power BI to analyze sales performance across different mobile brands, cities, payment methods, and time periods. The dashboard provides key business insights through dynamic visualizations, KPIs, and drill-down capabilities, enabling data-driven decision-making.

Key Features:
Sales Performance Analysis
Total Sales: 769M
Total Quantity Sold: 19K
Total Transactions: 4K
Average Selling Price: 40.11K
Interactive Filters (Slicers)
Mobile Model
Payment Method
Brand
Year, Quarter, Month, and Day
Visualizations
Sales distribution by city (Map Visualization)
Monthly quantity trend analysis
Sales by day of the week
Sales by mobile model
Transaction analysis by payment method
Brand-wise sales and transaction comparison
Month-to-Date (MTD) performance tracking
Same Period Last Year (SPLY) comparison
Quarterly and Monthly sales trend analysis
Business Insights
Identifies top-performing brands and cities
Tracks sales growth over time
Compares current performance against previous years
Analyzes customer payment preferences
Monitors sales KPIs in real time
Tools & Technologies:
Power BI
DAX (Data Analysis Expressions)
Power Query
Data Modeling
Interactive Dashboard Design

This project demonstrates expertise in data visualization, business intelligence, KPI reporting, dashboard development, and analytical storytelling using Power BI.
